Synthesizer¶

class Electronics.Instruments.Synthesizer¶

Bases: support.NamedClass

Synthesizer prototype

It has only one output. So a more sophisticated device having multiple outputs must be subclassed as multiple one-output devices. Each of these must also be subclassed to this superclass. The helper method SynthesizerInstance() is used to provide the correct instance and to make sure that there is only one, so that there is no conflict in the device use.
